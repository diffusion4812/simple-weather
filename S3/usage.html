Simple Weather Service API Reference

This service provides weather information for given weather station IDs:
Station ID Location
SILJ St. Ives, Cornwall
CQPY Penryn, Cornwall
ZVDB St. Just, Cornwall
TLFO Truro, Cornwall
URNJ Redruth, Cornwall

Usage:

Responses:

HTTP 200 - success
Schema (application/json):


            {
                "temperature": (temperature in Celsius)
            }

HTTP 400 - failure
Schema (application/json):


            {
                "error": "Request failed"
            }

Responses:

HTTP 200 - success
Schema (application/json):


            {
                "speed": (wind speed in m/s),
                "bearing": (wind bearing in degrees - wind is heading in THIS direction)
            }

HTTP 400 - failure
Schema (application/json):


            {
                "error": "Request failed"
            }

Responses:

HTTP 200 - success
Schema (application/json):


            {
                "humidity": (percentage humidity)
            }

HTTP 400 - failure
Schema (application/json):


            {
                "error": "Request failed"
            }

Responses:

HTTP 200 - success
Schema (application/json):


            {
                "pressure": (pressure in hPa)
            }

HTTP 400 - failure
Schema (application/json):


            {
                "error": "Request failed"
            }
